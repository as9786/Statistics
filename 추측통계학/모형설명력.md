- Goodness of fit
- 설명력이 높다 = 종속변수를 잘 예측한 것
- $y_i$ : 관측값, $\bar y$ : 관측값의 평균, $\hat y_i$ : 예측값
1. SST(Total Sum of Squares) = $\sum_{i=1}^N (y_i - \bar y)^2$
  - 흩어진 정도
2. SSE(Explained Sum of Squares) = $\sum_{i=1}^N(\hat y_i - \bar y)^2$
  - 모형이 설명하는 정도
  - $\hat y_i = \hat b_0 + \hat b_1 X_1$
3. SSR(Residual Sum of Squares) = $\sum_{i=1}^N (y_i - \hat y_i)^2$
  - 잔차
  - 모형이 설명하지 못하는 정도
- SST = SSE + SSR (E(u|x)=0을 만족해야 함)

# 결정계수
- $0 \leq R^2 = \frac{SSE}{SST} \leq 1$
- 독립변수의 수가 증가하면 결정계수는 일반적으로 증가. 설령 설명력이 없는 독립변수가 추가되어도 증가
- y가 가지고 있는 전체 분산 중에서 모형에 의해서 설명되어지는 정도를 의미
- 모형 설명력의 통계적 유의미성
- 상수항만 있는 모형에 비해서 얼마나 더 잘 설명하는지, 그 차이가 통계적으로 유의미한지
- 귀무가설 : 상수항만 있는 모형의 설명력과 동일하다 -> F-Test
## 가설검정
- 모집단에서 상수항만 존재하는 모형의 설명력과 비교
- 분산 분석 방법 사용
- y의 전체 분산 = 모형에 의해서 설명되는 분산 + 모형에 의해서 설명되지 못하는 분산
- 모형의 설명력이 좋을수록, 모형에 의해 설명되는 분산이 큼
- 모형에 의해서 설명되는 분산 / 설명되지 못하는 분산
- 분산에 대한 추정식 => $F=MS_M/MS_R$ -> F-Distribution(크면 클수록 기각확률 증가, 기각 영역은 오른쪽에만)
- $MS_M = SSE/df_M$
- $MS_R = SSR/df_R$
- $df_M$ : 독립변수의 수, $df_R$ : 표본 크기 - 모수의 수

## 조정된 결정계수($Adjust R^2$)
- 결정계수는 독립변수의 수가 증가하면 절대 감소하지 않음
- $adj\ R^2 = 1 - (1-R^2)[\frac{N-1}{N-(k+1)}]$
- k : 독립변수의 수, N : 표본 크기
