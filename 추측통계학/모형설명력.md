- Goodness of fit
- 설명력이 높다 = 종속변수를 잘 예측한 것
- $y_i$ : 관측값, $\bar y$ : 관측값의 평균, $\hat y_i$ : 예측값
1. SST(Total Sum of Squares) = $\sum_{i=1}^N (y_i - \bar y)^2$
  - 흩어진 정도
2. SSE(Explained Sum of Squares) = $\sum_{i=1}^N(\hat y_i - \bar y)^2$
  - 모형이 설명하는 정도
  - $\hat y_i = \hat b_0 + \hat b_1 X_1$
3. SSR(Residual Sum of Squares) = $\sum_{i=1}^N (y_i - \hat y_i)^2$
  - 잔차
  - 모형이 설명하지 못하는 정도
- SST = SSE + SSR (E(u|x)=0을 만족해야 함)
- $0 \leq R^2 = \frac{SSE}{SST} \leq 1$
