# Gibbs sampling
- Metropolis weakness : 시간이 오래 걸림
- 보통 모수가 여러 개인 경우 적용
- 예시 : 서로 다른 두 개의 동전
- 가장 먼저 가능도와 사전 분포를 정해야 함
- 관찰치는 서로 독립이라고 가정
- 사전 분포는 두 모수에 대한 결합 확률이 되어야 함
- 각 변수의 조건부 분포를 번갈아 가면서 표분 추출하여 전체 분포를 따르도록
- p(x,y)는 구하기 어렵지만, p(x|y), p(y|x)를 이용할 수 있을 때

## 작동 방식
- 기본적으로 metropolis와 유사
- 새로운 값이 선택되는 것이 다름
- 여러 개 모수 중에서 하나가 선택
- 조건부 사후 확률분포를 이용해서 결정 : $p(\theta_i|\theta_{j \neq i}, D)$
- x 고정 -> y 표본 추출
- y 고정 -> x 표본 추출
- 초기 : $(x_0, y_0)$
- $x_{t+1} \sim p(x|y_t)$
- $y_{t+1} \sim p(y|x_{t+1})$
