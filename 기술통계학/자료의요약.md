- 정보를 하나의 요약수치(summary measure)로 나타내면 자료에 대한 보다 정확한 분석 가능

## 1. 집중경향치
- 자료의 중심을 측정하는 척도
- 평균, 중앙값, 최빈값
- (가)에서 A의 집중경향치가 B의 집중경향치보다 작음

## 2. 산포도
- 관측값들이 얼마나 좁게 또는 넓게 분포되어 있는지를(자료의 흩어진 정도) 나타내는 척도
- 범위, 분산, 표준편차, 변동계수
- (나)에서 A가 B보다 넓게 분포되어 있음

## 3. 비대칭도
- 분포의 정점에서 수직선을 그어 좌우가 같으면 대칭(symmetric)인데 그렇지 않고 분포가 왼쪽이나 오른쪽으로 치우쳐 있을 때 그 정도를 나타내는 것
- (다)에서 A는 오른쪽 꼬리분포, B는 왼쪽 꼬리분포

## 4. 첨도
- 분포의 모양이 얼마나 뾰족한가
- 높은 첨도(leptokurtic), 보통 첨도(mesokurtic), 낮은 첨도(platykurtic)

## 1. 정리되지 않은 자료

### 1. 산술평균(Arithmetic mean)
- 분포의 중심을 나타내는 수치
- 수량자료에만 적용
- 자료의 값을 더해서 자료의 수로 나눈 값

![render](https://user-images.githubusercontent.com/80622859/180453852-0af1217c-2f30-4102-b58d-356ab544c5ef.png)

### 2. 중앙값(Median)
- 분포의 중심을 나타내는 수치, 수량자료에만 적용
- 자료의 수가 홀수이면 한가운데 위치한 수, 짝수이면 가운데 있는 두 수의 평균

### 통합표뵨평균

![render (1)](https://user-images.githubusercontent.com/80622859/180454609-c4011af7-2f87-48c8-ad70-701d5a90ab12.png)

- 각 집단의 평균을 각 집단의 상대적인 크기로 가중평균한 수치

### 3. 최빈값(Mode)
- 자료 중에서 발생빈도가 가장 높은 관측값
- 범주형 자료에도 적용

### 4. 자료의 분포 모양
- 평균 != 중앙값 => 비대칭
- 오른쪽 꼬리 분포 : 평균 > 중앙값
- 왼쪽 꼬리 분포 : 평균 < 중앙값
- 평균은 극단값의 영향에 민감, 중앙값은 영향을 받지 않음
- 최빈값 : 소비자들이 자주 요구하는 것이 무엇인지 파악
- 중앙값 : 사회과학에서 비대칭분포의 대표값으로 많이 사용, ex) 중위소득(median income)
- 평균 : 통계적 추측에서 표본의 대표값

# 3. 상자-수염 그림
- Boxplot
- 두 그룹의 분포를 비교하는데 효과적
- 그리는 방법
1) 1사분위수와 3사분위수의 값을 양쪽 끝부분(hinge)으로 하여 박스를 그림. 박스의 폭은 (Q3-Q1) = IQR(interquartile range)
2) 박스 안으로 중앙값에 해당하는 위치에 선을 그음
3) 박스의 양쪽 끝으로부터 시작하여 박스의 바깥쪽 방향으로 선을 그음(whiskers). 이 선의 길이는 1.5 x IQR
4) 어떤 관측치가 박스의 양쪽 끝으로부터 3 x IQR 밖의 범위에 존재하면 극단 이상치(extreme outliers), 1.5 x IQR과 3 x IQR 사이는 보통 이상치(mild outliers라 하며 이는 0으로 표시

# 4. 평균과 표준편차와의 관계

## 1. Z값(Z-value)
- 특정 관측값(xi)이 자료의 평균으로부터 몇 개의 표준편차만큼 떨어져 있는지를 나타내는 상대적 위치의 척도(measure of relative standing)

![캡처](https://user-images.githubusercontent.com/80622859/180460138-1eb8190e-790d-4b2d-b137-c4342efce763.PNG)

## 2. 체비셰프의 법칙(Chebyshev's Inequality)
-  자료의 분포모양에 상관없이 적어도 관측치들의 (1-1/k^2)x100(%)가 평균으로부터 +k, -k 개의 표준편차 내에 위치한다는 정리
-  k는 1보다 큰 상수

## 3. 경험에 의한 법칙(Rule of Thumb, Normal Approximation)
- 자료의 분포모양이 대략적으로 평균을 중심으로 좌우대칭이며, 종모양과 비슷할 시
1. 관측값들의 약 68%가 평균으로부터 +-1 개의 표준편차내에 위치
2. 관측치값들의 약 95%가 평균으로부터 +-2 개의 표준편차내에 위치
3. 관측치값들의 약 99% 이상이 평균으로부터 +-3 개의 표준편차내에 위치

# 5. 비대칭도와 첨도
- 분포의 모양을 측정하는 도구(measure of shape)

## 1. 비대칭도(skewness)

### 1. 비대칭도의 측정
- 평균과 중앙값 비교
- 피어슨 비대칭계수(Pearson's coefficient of skewness, Pearson's CS)

![캡처](https://user-images.githubusercontent.com/80622859/180461353-27452407-3691-4c66-afcb-48b578fdf70d.PNG)

- 평균이 중앙값보다 크면 위의 값이 양, 반대의 경우 음의 값을 갖게 됨
- 평균과 중앙값의 차이가 클수록 비대칭계수의 절대값 크기는 크게됨
- 0에 가까울수록 분포의 모양은 좌우대칭에 가까움

## 2. 첨도(Kurtosis)
- 분포의 뾰족한 정도를 측정
- 보통 첨도 : 첨도계수(coefficient of kurtosis) = 3, 정규분포(normal distribution)의 첨도
- 높은 첨도 : 첨도계수 > 3
- 낮은 첨도 : 첨도계수 < 3
- 첨도계수는 첨도가 정의된다면 항상 1 이상의 값을 가짐
