# 1. 정의
- 오차의 분산이 모든 관측치에서 동일
- $Var(\epsilon_i | X) = \sigma^2$ 
- i가 바뀌어도 오차의 분산은 항상 같은 값
- 모든 구간에서 퍼짐 정도가 비슷
- 이분산 : 어떤 구간은 분산이 작고, 어떤 구간은 분산이 큰 경우
- 등분산 가정을 안하게 될 경우, 분산이 틀어져 신뢰 구간이 다 틀리게 됨 => 통계적 검정 무의미

# 2. White test

## 2-1. 이분산성

- 이분산성(Heteroskedasticity)이 존재하는지
- 이분산성 : 오차항의 분산이 일정 X
- $Var(\epsilon_i) \neq \sigma^2$
- 어떤 구간에서는 오차가 크고, 어떤 구간에서는 오차가 작다

## 2-2. 검정
- 귀무가설 : 등분산성(Homoskedasticity)
- 대립가설 : 이분상성

1. 회귀 모형
  - $y=\beta_0 + \beta_1 x_1 + \cdots + \epsilon$
2. 잔차의 제곱을 종속 변수로 사용
  - $\hat\epsilon^2 = \alpha_0 + \alpha_1 x_1 + \alpha_2 + x_1^2+\cdots$
